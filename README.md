# multi-ai-workflow-game-development
📌 Overview
Not just a game, but an exploration of how AI actually builds one. 
This project uses Google Gemini to design prompts and evaluates how ChatGPT and Claude translate those prompts into working code revealing strengths, gaps, and the real role of human input in AI-assisted development.

🎯 Objective
To evaluate the effectiveness of AI tools in generating working code
To compare outputs from different AI models
To understand the role of prompt engineering in improving results

🧠 AI Workflow Used
Prompt generated using Google Gemini
Code generated using ChatGPT and Claude
Outputs analyzed and refined manually

🛠️ Tools Used
Google Gemini (Prompt generation)
ChatGPT (Code generation)
Claude (Code generation)
HTML5 Canvas, JavaScript, CSS

📂 Project Structure
/final-game/         # Final working version
/chatgpt-version/   # Code generated using ChatGPT
/claude-version/    # Code generated using Claude
/prompts/           # All prompts used (Gemini + iterations)
